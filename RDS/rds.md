# What is AWS RDS ?

AWS RDS (Relational Database Service) is a managed database service by AWS.

In simple words:

> RDS lets you run a SQL database in the cloud without managing servers, backups, or updates.

AWS handles:
- Database setup
- Backups
- Security patches
- Scaling
- Monitoring

## Supported Databases
- MySQL
- PostgreSQL
- MariaDB
- Oracle
- SQL Server
- Aurora

## Key Features
- **Automated Backups** → Easy data recovery  
- **Multi-AZ** → High availability  
- **Read Replicas** → Better read performance  
- **Security** → Encryption and access control  

## When to Use RDS
Use RDS when your application needs structured data such as:
- User data
- Orders
- Payments

## Summary
> AWS RDS is a managed SQL database service where AWS manages infrastructure, and you manage your data.
